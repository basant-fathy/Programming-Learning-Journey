# 10 - OOP as It Should Be (Concepts)

This course focuses on building **correct Object-Oriented Programming (OOP) thinking**.  
It explains *why* OOP exists, *when* to use it, and *how* to apply it properly without misuse or overengineering.

The course emphasizes understanding concepts before implementation.
